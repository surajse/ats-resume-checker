# **App Name**: ResumeAce

## Core Features:

- ATS Score Calculation: Calculate an ATS compatibility score based on keyword analysis, formatting, and structure.
- Keyword Match Analysis: Extract keywords from a job description and compare them against the resume text.
- ATS Rule Checks: Apply a set of predefined rules to check for common ATS formatting and content issues.
- Formatting Issue Detection: Identify potential formatting issues such as excessive symbols or multiple fonts.
- Bullet Point Improvement (AI-Suggested): Suggest improvements to bullet points, transforming weak statements into action-oriented descriptions with skill and impact. This feature uses a predefined tool that incorporates relevant improvements based on the textual context.
- Optimized ATS Resume Output: Generate a plain text, ATS-friendly version of the resume with clean sections and optimized keywords.
- User Data Persistence: Allow the user to save the last 3 resume analyses.

## Style Guidelines:

- Primary color: Dark slate blue (#37474F) for a professional and trustworthy feel.
- Background color: Light gray (#ECEFF1) for a clean and modern look.
- Accent color: Teal (#009688) for highlights and call-to-action buttons.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral, objective feel.
- Use simple, professional icons for different sections and functions.
- Single-page layout with clear sections for input, analysis, and output.
- Subtle transitions and animations for feedback and loading states.
